# 开源全量包：极简使用说明
version: 1.0.0

这个包已经包含：

- 一份可直接运行的项目模板（`project-template/`）
- 两个技能（`skills/`）
  - `zhongdong-congling-gongcheng`：从零建站 + 更新流程
  - `zhongdong-zongskill-anzhuang-diaoyong`：安装与调用总流程

## 安全说明（默认防误操作）

- 本包不包含 `.git`，也不包含任何账号、密码、Token、SSH密钥。
- 默认只做本地更新，不会自动发布。
- 只有你明确下达“发布”指令时，AI才应进入发布步骤。

## 最少操作（建议）

1. 把 `project-template/` 复制到你自己的工作目录（可改名）。
2. 执行技能安装脚本：
   - macOS/Linux：`bash 安装技能.sh`
   - Windows：双击 `安装技能.bat`
3. 新开AI会话后，输入：

```text
请使用 $zhongdong-zongskill-anzhuang-diaoyong，先检查安装状态，再从零初始化并执行一次本地全量更新，不要发布。
```

## 目录说明

- `project-template/`：网站项目模板
- `skills/`：可安装的技能目录
- `安装技能.sh` / `安装技能.bat`：一键复制技能到默认目录
- `调用示例.txt`：可直接复制给AI的触发词
