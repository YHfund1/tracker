# 从零初始化

## 目标

在没有现成工程的情况下，先创建一套可持续更新的中东地缘跟踪网站骨架。

## 推荐目录结构

```text
project-root/
  index.html
  briefing.html
  war-situation.html
  news.html
  central-bank-tracker.html
  research.html
  oil-chart.html
  data-tracking.html
  eco-track.html
  polymarket.html

  data/
    supply-chain.json
    cb-statements.json
    fedwatch.json
    polymarket_data.json
    eco_data.json
    excel_history.json
    research_raw_data.json

  scripts/
    jin10_fetch.py
    fetch_fedwatch.py
    update_research_data.py

  update_strait_data.py
  update_data_from_excel.py
  scrape_cls_final.py
  update_polymarket_html.py
  generate_timelapse_video.py
```

## 初始化步骤

1. 创建目录与空文件（或最小可运行模板）。
2. 为每个页面保留统一导航与“最后更新时间”展示位。
3. 让以下页面默认改为读取 JSON（避免把大量数据硬编码进 HTML）：
   - `index.html` 读取 `data/supply-chain.json`
   - `central-bank-tracker.html` 读取 `data/cb-statements.json`、`data/fedwatch.json`
   - `data-tracking.html` 读取 `data/excel_history.json`
   - `eco-track.html` 读取 `data/eco_data.json`
   - `polymarket.html` 读取 `data/polymarket_data.json`
4. 为 `scripts/` 预留可执行脚本入口，命名保持稳定，方便后续自动化。

## 最小可用标准

- 打开任一页面不会报错。
- 缺数据时页面有空态提示，不白屏。
- 所有 JSON 文件是合法 JSON。
