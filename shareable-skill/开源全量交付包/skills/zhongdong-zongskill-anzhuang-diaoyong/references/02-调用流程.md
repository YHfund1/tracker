# 调用流程

## 推荐调用顺序

1. 先调用总技能：完成安装检查与任务编排。
2. 再由总技能调度子技能：执行从零搭建或全量更新。

## 推荐触发词

### 触发总技能

```text
请使用 $zhongdong-zongskill-anzhuang-diaoyong，带我从零搭建并执行一次中东地缘跟踪网站全量更新，先本地验收，不要发布。
```

### 触发子技能（直接）

```text
请使用 $zhongdong-congling-gongcheng，从零创建一个中东地缘跟踪工程，并按标准流程完成本地全量更新与校验。
```

## 输出要求

每次执行后应返回：

- 已完成步骤清单
- 已更新文件清单
- 未完成项与原因
- 是否触发了发布（默认应为否）
