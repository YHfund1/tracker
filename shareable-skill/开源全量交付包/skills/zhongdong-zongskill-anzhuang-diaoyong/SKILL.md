# 中东总技能（安装与调用）

## 适用场景

- 你要把技能包发给别人，并确保对方 AI 能直接执行。
- 对方是全新环境，没有既有工程。
- 需要一套统一的安装、调用、验收流程。

## 总技能职责

1. 按 [references/01-安装流程.md](references/01-安装流程.md) 指导对方安装技能包。
2. 按 [references/02-调用流程.md](references/02-调用流程.md) 给出稳定触发词与执行顺序。
3. 按 [references/03-从零落地流程.md](references/03-从零落地流程.md) 指导对方从零建工程。
4. 按 [references/04-安全发布边界.md](references/04-安全发布边界.md) 控制误操作风险。

## 默认执行原则

- 先安装、后验证、再调用。
- 先本地、后发布。
- 先确认目录与数据结构，再运行更新。
- 未明确授权，不做任何远程写入操作。
